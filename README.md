# NameNest MVP

Trilingual baby-name site MVP for Portuguese, English, and Spanish.

## Features

- Homepage for `/`, `/pt`, `/en`, `/es`
- Baby-name generator with gender, style, and language fit filters
- Family honor-name tool for grandparents/relatives
- Static name detail pages in PT/EN/ES
- Starter curated name database
- Vitest tests for generator/search/family honor logic

## Commands

```bash
npm install
npm test
npm run build
npm run dev
```
